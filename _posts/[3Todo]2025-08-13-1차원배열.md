## 8. Array vs List vs ArrayList vs Set(HashSet)
Array(배열): 길이 고정, 원시 타입 가능, 인덱스 O(1)

List: 순서/중복 허용, 인덱스 접근 가능

ArrayList: 가변 길이 배열 기반 List

Set: 중복 없음, 인덱스 없음

HashSet: 해시 기반 Set(순서X)


    List<String> L = new ArrayList<>();
    L.add("a");
    L.get(0);
    int i = L.indexOf("a");

    Set<Integer> S = new HashSet<>();
    S.add(10);
    S.contains(10);

// Stream 사용
